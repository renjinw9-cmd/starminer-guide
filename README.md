# Starminer Guide

Fan-made strategy guide for **Starminer** — the deep-space industrial mining simulator by CoolAndGoodGames (Steam Early Access, May 2026).

🌐 **Live at:** [starminer-guide.vercel.app](https://starminer-guide.vercel.app)

## Pages

- **Beginner Guide** — First hour, controls, survival loop, essential tips
- **Mining & Resources** — Complete resource database, refinery tiers, Drill Node automation
- **Ship Building** — Mass management, thruster placement, modular design, blueprints
- **Research & Tech Tree** — Optimal unlock order, research lab optimization
- **Combat & Defense** — Weapons, shields, heat management, enemy tactics

## Tech

- Static HTML/CSS/JS — no framework, no build step
- Deployed on Vercel (automatic from `main` branch)
